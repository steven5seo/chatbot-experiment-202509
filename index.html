LLM Chatbot Demo

A lightweight web UI for chatting with any OpenAI-compatible LLM API. Supply your own API key, tweak settings, and explore responses.

About this demo

This static site is designed for GitHub Pages and works with any API that follows the OpenAI Chat Completions protocol. Your API key is never stored on our servers — all requests are made directly from your browser.

Usage tips
  • Use the settings panel above to paste your API key, change the base URL for self-hosted gateways, or switch to a different model.
  • Toggle the sun/moon button to switch between light and dark themes. The choice persists in localStorage.
  • Click "Reset" in the settings section to clear saved configuration, including the remembered API key.
